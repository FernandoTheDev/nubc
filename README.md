# nubc

A compiler for the [Nub Programming Language](https://codeberg.org/nublang/nub).

> [!WARNING]
> This project is in its very early stages. Both the compiler and the language specification are highly unstable — breaking changes may occur at any time.

## Status

`nubc` is currently pre-alpha. Core features are missing or incomplete, and the language itself is still being designed. Use at your own risk.
